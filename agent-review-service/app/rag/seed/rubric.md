# 初中作文批改质量评估 Rubric（六维度）

> 评估对象：AI 大模型对中学生作文的**批改结果**（而非作文本身）。

## 1. 主旨（theme）

| 分数 | 标准 |
| --- | --- |
| 5 | 准确指出作文中心思想，评析题意契合度，点出主题表达的深浅 |
| 4 | 基本指出中心，有一定题意分析 |
| 3 | 提及主题但浮于表面 |
| 2 | 主题判断错误或混淆 |
| 1 | 完全没有触及主题 |

典型问题：只复述作文内容不做主题评析、把记叙文当作议论文评价、脱离题目要求泛泛而谈。

## 2. 想象（imagination）

| 分数 | 标准 |
| --- | --- |
| 5 | 肯定合理想象、指出创意闪光点、鼓励独特视角；对不合常识的想象给予温和纠正 |
| 4 | 识别作文中的想象元素并给出评价 |
| 3 | 对想象力泛泛而谈 |
| 2 | 忽视作文里的想象性段落 |
| 1 | 错误地把想象判为偏题 |

## 3. 逻辑（logic）

| 分数 | 标准 |
| --- | --- |
| 5 | 分析结构顺序、段落过渡、首尾呼应、主次详略得当 |
| 4 | 点评总体结构 |
| 3 | 只关注是否分段 |
| 2 | 结构分析错误 |
| 1 | 缺少逻辑层面的评价 |

## 4. 语言（language）

| 分数 | 标准 |
| --- | --- |
| 5 | 识别典型好句（比喻/拟人/排比）、点出病句与用词不当、给出改写建议 |
| 4 | 指出部分语言问题 |
| 3 | 只做笼统评价（"语言流畅"） |
| 2 | 错改病句或误判好句 |
| 1 | 无语言维度评析 |

## 5. 书写（writing）

| 分数 | 标准 |
| --- | --- |
| 5 | 评价书写工整度、字迹清晰度、标点规范、卷面整洁；基于图片客观判断 |
| 4 | 整体点评 |
| 3 | 简略提及 |
| 2 | 凭空猜测书写情况 |
| 1 | 完全忽视书写维度 |

## 6. 整体评价（overall）

| 分数 | 标准 |
| --- | --- |
| 5 | 总评清晰，既有肯定也有建设性建议，分数/等级判断合理 |
| 4 | 总结到位 |
| 3 | 流于程式化"加油"式评语 |
| 2 | 评价与前面维度自相矛盾 |
| 1 | 没有整体小结 |

> **决策规则**：在两份批改比较时，整体评价维度直接决定 winner。
